* Built with Spring Boot and Angular 2, using a microservice architecture to give faster response times to the user.

* Gives users the ability to create a household to synchronize calendars and to-do lists

* Includes a fully customizable, dashboard like interface for the user to view their household's information

* Collaborated with 3 other groups members to design, implement, and test features in the application
